Install the library, provide your price data, and get indicator results with just a few lines of C#. This guide walks through every step from package installation to iterating results.

<Steps>
  <Step title="Install the NuGet package">
    Add `Skender.Stock.Indicators` to your .NET project using your preferred tool.

    <CodeGroup>

    ```bash dotnet CLI
    dotnet add package Skender.Stock.Indicators
    ```

    ```powershell Package Manager
    Install-Package Skender.Stock.Indicators
    ```

    </CodeGroup>

    The package targets .NET 8, 9, and 10. No additional dependencies are required.
  </Step>

  <Step title="Provide historical quote data">
    Most indicators require a series of OHLCV (Open, High, Low, Close, Volume) price quotes. The library does not supply market data — you must fetch it from your own source.

    Quotes are provided as an `IReadOnlyList<Quote>`, `List<Quote>`, or `ICollection<Quote>`. Each `Quote` has six properties:

    | Property    | Type       | Description  |
    | ----------- | ---------- | ------------ |
    | `Timestamp` | `DateTime` | Close date   |
    | `Open`      | `decimal`  | Open price   |
    | `High`      | `decimal`  | High price   |
    | `Low`       | `decimal`  | Low price    |
    | `Close`     | `decimal`  | Close price  |
    | `Volume`    | `decimal`  | Volume       |

    ```csharp
    using Skender.Stock.Indicators;

    // fetch quotes from your own data source
    IReadOnlyList<Quote> quotes = GetQuotesFromFeed("MSFT");
    ```

    <Tip>
      As a general rule, provide at least 750 data points (roughly 3 years of daily data). Many indicators use smoothing algorithms that require more history than the stated minimum to converge to accurate values.
    </Tip>
  </Step>

  <Step title="Calculate an indicator">
    Call the extension method for the indicator you need. Pass any required parameters — default values are provided where an industry standard exists.

    ```csharp
    // calculate 20-period Simple Moving Average
    IReadOnlyList<SmaResult> results = quotes.ToSma(20);
    ```

    Every indicator follows the same pattern: `quotes.To{IndicatorName}(parameters)`. The result is always an `IReadOnlyList<T>` aligned to the input quote series.
  </Step>

  <Step title="Iterate the results">
    Each result entry corresponds to one quote period and contains a `Timestamp` plus the indicator-specific output values. Values are `null` for periods in the warmup window before enough history exists to calculate.

    ```csharp
    foreach (SmaResult r in results)
    {
        Console.WriteLine($"SMA on {r.Timestamp:d} was ${r.Sma:N4}");
    }
    ```

    Example output:

    ```console
    SMA on 4/19/2018 was $255.0590
    SMA on 4/20/2018 was $255.2015
    SMA on 4/23/2018 was $255.6135
    SMA on 4/24/2018 was $255.5105
    SMA on 4/25/2018 was $255.6570
    SMA on 4/26/2018 was $255.9705
    ..
    ```
  </Step>

  <Step title="Remove warmup periods (optional)">
    Results include early entries where the indicator has not yet accumulated enough data. Filter these out with `RemoveWarmupPeriods()` when you only want fully-calculated values.

    ```csharp
    IReadOnlyList<SmaResult> results = quotes
        .ToSma(20)
        .RemoveWarmupPeriods();
    ```

    <Warning>
      Do not rely on the minimum quote requirement as an optimization target. Smoothed indicators (EMA, ATR, RSI, etc.) converge to precise values only after significantly more history than the technical minimum. Provide as much history as you have and let `RemoveWarmupPeriods()` trim the output.
    </Warning>
  </Step>
</Steps>

## Complete example

```csharp
using Skender.Stock.Indicators;

// step 1: fetch historical quotes from your data source
IReadOnlyList<Quote> quotes = GetQuotesFromFeed("MSFT");

// step 2: calculate 20-period SMA
IReadOnlyList<SmaResult> results = quotes
    .ToSma(20)
    .RemoveWarmupPeriods();

// step 3: use the results
foreach (SmaResult r in results)
{
    Console.WriteLine($"SMA on {r.Timestamp:d} was ${r.Sma:N4}");
}
```
