Stock Indicators for .NET is an open-source C# library that turns raw market price data into actionable technical indicators. Install a single NuGet package and get access to 84+ battle-tested indicators — from simple moving averages to Ichimoku clouds — with support for batch, incremental, and real-time streaming workflows.

## Get up and running

<Steps>
  <Step title="Install the NuGet package">
    Add `Skender.Stock.Indicators` to your .NET project using the CLI or Package Manager.

    ```bash
    dotnet add package Skender.Stock.Indicators
    ```
  </Step>
  <Step title="Provide historical quotes">
    Supply OHLCV price data as `IReadOnlyList<Quote>` from your market data provider.

    ```csharp
    using Skender.Stock.Indicators;

    IReadOnlyList<Quote> quotes = GetQuotesFromFeed("MSFT");
    ```
  </Step>
  <Step title="Calculate an indicator">
    Call any `To{Indicator}()` extension method on your quotes collection.

    ```csharp
    IReadOnlyList<SmaResult> results = quotes.ToSma(20);
    ```
  </Step>
  <Step title="Use the results">
    Iterate over the typed results for charting, alerting, or trading logic.

    ```csharp
    foreach (SmaResult r in results)
    {
        Console.WriteLine($"SMA on {r.Timestamp:d} was {r.Sma:N4}");
    }
    ```
  </Step>
</Steps>
