---
title: "VWMA: Volume Weighted Moving Average for Stock.Indicators"
description: "The VWMA weights each price by traded volume, highlighting price levels with the heaviest participation and helping confirm trend strength and direction."
---

The Volume Weighted Moving Average computes the average price over a lookback window where each bar's contribution is proportional to its trading volume. Bars with heavy volume pull the average toward their price level, while low-volume bars have less influence. This makes VWMA more informative than a simple average in markets where volume is unevenly distributed across the day or session. It is commonly used to confirm trend strength and to identify institutional participation in price moves.

<Note>
  VWMA requires both price and volume data from your quote source. It cannot be generated by chaining from another indicator's output — it must start from raw `IQuote` data.
</Note>

## Parameters

<ParamField path="lookbackPeriods" type="int" required>
  Number of periods (`N`) in the moving average. Must be greater than 0.
</ParamField>

### Minimum history

You must supply at least `N` periods of quotes to cover the warmup period.

## Result fields

<ResponseField name="Timestamp" type="DateTime" required>
  Date from the evaluated quote.
</ResponseField>

<ResponseField name="Vwma" type="double">
  Volume Weighted Moving Average value. Returns `null` for the first `N-1` periods.
</ResponseField>

## Usage

```csharp
// batch series (requires IQuote with Volume data)
IReadOnlyList<VwmaResult> results =
    quotes.ToVwma(lookbackPeriods);
```

## Streaming

Use `VwmaList` for incremental calculations without a hub:

```csharp
VwmaList vwmaList = new(lookbackPeriods);

foreach (IQuote quote in quotes)  // simulating stream
{
    vwmaList.Add(quote);
}

// based on ICollection<VwmaResult>
IReadOnlyList<VwmaResult> results = vwmaList;
```

`VwmaList` also accepts direct price and volume inputs:

```csharp
VwmaList vwmaList = new(lookbackPeriods);

// add individual price and volume data points
vwmaList.Add(DateTime.Now, price: 100.50, volume: 1000);
```

Subscribe to a `QuoteHub` for advanced streaming scenarios:

```csharp
QuoteHub quoteHub = new();
VwmaHub observer = quoteHub.ToVwmaHub(lookbackPeriods);

foreach (IQuote quote in quotes)  // simulating stream
{
    quoteHub.Add(quote);
}

IReadOnlyList<VwmaResult> results = observer.Results;
```

See [buffer lists](/guide/buffer) and [stream hubs](/guide/stream) for full usage guides.

## Chaining

VWMA results implement `IReusable` and can be chained into downstream indicators:

```csharp
// feed VWMA output into RSI
var results = quotes
    .ToVwma(lookbackPeriods)
    .ToRsi(rsiPeriods);
```

<Warning>
  VWMA must be generated from `quotes` directly. It **cannot** be generated from the results of another chain-enabled indicator because volume data is required at the source.
</Warning>

See [chaining indicators](/guide/batch#chaining-indicators) for more.
